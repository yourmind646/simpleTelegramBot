# simpleTelegramBot

Тестовый проект для обучения Go: Telegram-бот с зачатками “игры” (меню/кнопки) и хранением состояния пользователя в Redis.

## Что уже есть
- Обработка апдейтов через роутер (`handlers/`) и набор маршрутов (check/action).
- Команды `/start` и `/help`, главное меню с reply-клавиатурой (`keyboards/`).
- FSM на Redis для хранения состояния пользователя (`fsm/`).

## Зависимости
- Go (версия указана в `go.mod`).
- Redis на `localhost:6379` (без пароля, DB 0 — см. `main.go`).

## Запуск
1) Создай бота у @BotFather и возьми токен.
2) Подними Redis (пример через Docker):
   ```bash
   docker run --rm -p 6379:6379 redis:7
   ```
3) Укажи токен в переменной окружения и запусти:
   ```bash
   export TELEGRAM_APITOKEN="123456:ABCDEF..."
   go run .
   ```

## Структура
- `main.go` — инициализация бота, получение апдейтов, запуск роутинга.
- `handlers/` — маршруты и обработчики.
- `keyboards/` — клавиатуры Telegram.
- `fsm/` — хранение/получение состояния пользователя (Redis).
